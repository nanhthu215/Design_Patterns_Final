🔐 SINGLETON PATTERN - What Was Implemented
════════════════════════════════════════════════════════════════════════════

✅ PRIVATE CONSTRUCTOR
   └─ Prevents direct instantiation: new CartService() ❌ throws error

✅ STATIC PRIVATE INSTANCE
   └─ static #instance = null;
   └─ Hidden from outside access

✅ getInstance() METHOD
   └─ First call: creates instance
   └─ Later calls: returns same instance
   └─ Guaranteed single instance

✅ STATIC METHODS ONLY
   └─ All methods are static
   └─ Operates on shared state
   └─ No instance methods needed

✅ OBSERVER PATTERN INTEGRATION
   └─ subscribe() to track changes
   └─ Auto-notify listeners on updates
   └─ Real-time UI synchronization

✅ PERSISTENCE
   └─ localStorage auto-save
   └─ Survives page refresh

✅ CROSS-TAB SYNC
   └─ Storage events listener
   └─ Real-time sync between tabs
   └─ One unified cart across browser

💼 KEY FEATURES
════════════════════════════════════════════════════════════════════════════

Core Methods:
├─ getInstance()           - Get single instance
├─ addToCart()            - Add item to cart
├─ removeFromCart()       - Remove item
├─ updateQty()            - Update quantity
├─ clearCart()            - Clear all items
├─ getItems()             - Get all items
├─ getCartSummary()       - Get total price/qty
├─ findItem()             - Find specific item
├─ updateItemVariant()    - Change item options
└─ subscribe()            - Listen to changes (Observer)

Observer Features:
├─ subscribe() - Add listener
├─ Automatic notifications on:
│  ├─ ITEM_ADDED
│  ├─ ITEM_REMOVED
│  ├─ QTY_UPDATED
│  ├─ VARIANT_UPDATED
│  ├─ CART_CLEARED
│  └─ CART_SYNCED (from other tab)
└─ Event format: { type, data }

🎯 USAGE EXAMPLES
════════════════════════════════════════════════════════════════════════════

1️⃣  GET SINGLETON INSTANCE
   const cart = CartService.getInstance();
   // Returns same instance every call

2️⃣  ADD TO CART
   CartService.addToCart({
     productId: 1,
     name: 'Coffee',
     price: 50000,
     qty: 1
   });

3️⃣  SUBSCRIBE TO CHANGES (Observer)
   CartService.subscribe(({ type, data }) => {
     console.log(`Cart updated: ${type}`);
     console.log('Items:', data);
   });

4️⃣  USE IN REACT (with hook)
   const { items, addToCart, totalPrice } = useCartSingleton();

5️⃣  VERIFY SINGLE INSTANCE
   const ref1 = CartService.getInstance();
   const ref2 = CartService.getInstance();
   console.log(ref1 === ref2); // true ✅
