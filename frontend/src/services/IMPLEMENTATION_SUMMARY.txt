What Makes It True Singleton?

1️⃣  PRIVATE CONSTRUCTOR
    ✅ Prevents direct instantiation
    ✅ Only getInstance() can create instance
    
    Code:
    constructor() {
      if (CartService.#instance !== null) {
        throw new Error('CartService is a Singleton');
      }
    }

2️⃣  PRIVATE STATIC INSTANCE
    ✅ Holds the single instance
    ✅ Hidden from outside access
    
    Code:
    static #instance = null;

3️⃣  getInstance() METHOD
    ✅ Returns same instance every time
    ✅ Creates instance on first call
    
    Code:
    static getInstance() {
      if (CartService.#instance === null) {
        CartService.#instance = new CartService();
      }
      return CartService.#instance;
    }

4️⃣  STATIC METHODS ONLY
    ✅ No instance methods needed
    ✅ Operates on shared state
    
    Usage:
    CartService.addToCart(...);    // Static method
    CartService.getItems();        // Static method
    CartService.clearCart();       // Static method

💡 HOW TO USE
════════════════════════════════════════════════════════════════════════════

1️⃣  IN REACT COMPONENTS (Recommended)

import { useCartSingleton } from '../hooks/useCartSingleton';

export function ShoppingCart() {
  // Use the hook - automatically connected to Singleton
  const { 
    items, 
    addToCart, 
    removeFromCart, 
    totalPrice,
    itemCount,
  } = useCartSingleton();

  return (
    <div>
      <h2>My Cart ({itemCount})</h2>
      <p>Total: {totalPrice} VND</p>
      {items.map(item => (
        <div key={item.key}>
          {item.name} x {item.qty}
          <button onClick={() => removeFromCart(item.key)}>
            Remove
          </button>
        </div>
      ))}
    </div>
  );
}

2️⃣  OUTSIDE REACT COMPONENTS

import CartService from '../services/CartService';

// Get the single instance
const cart = CartService.getInstance();

// Use cart methods
cart.addToCart({ productId: 1, name: 'Coffee', price: 50000 });
const items = cart.getItems();
const summary = cart.getCartSummary();

3️⃣  WITH OBSERVER PATTERN

import CartService from '../services/CartService';

// Subscribe to changes
const unsubscribe = CartService.subscribe(({ type, data }) => {
  console.log(`Cart updated: ${type}`);
  console.log('Items:', data);
});

// Later: unsubscribe
unsubscribe();
